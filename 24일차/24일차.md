    2. 안드로이드는 알아서 2~3주안에 잘 작업하기.  백/프런트는 초기 설정부터  
    3. Git 규칙 
        a. 커밋규칙
            1. 수정한 것 -> Fix
                ex) 타이핑 수정했다면 #Fix typo 이런식으로
            2. 추가한 것 -> ADD
                ex) #Add Header
            3. 삭제한 것 -> Delete
                ex) #Delete Modal
            4. 수정한 것 -> ReFactor
                ex) #ReFactor Header
            등등등
        하는 이유 -> 커밋을 작게작게하려고
